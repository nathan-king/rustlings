# Rustlings

This repository contains my solutions and notes from completing **Rustlings**, a set of small exercises designed to teach core Rust concepts through practice.

## Topics Covered

Rustlings includes exercises on:

- Variables and mutability
- Ownership and borrowing
- References and lifetimes
- Structs and enums
- Pattern matching
- Error handling
- Modules and the Rust workspace system
- Testing
- Traits, generics, and iterators

## How to Run Rustlings

To install and run Rustlings on your machine:

```bash
git clone https://github.com/rust-lang/rustlings
cd rustlings
cargo install --force rustlings
rustlings watch
```
